# Contributing to Shenqu Algorithm

We welcome contributions from the community! Here's how you can help:

## Reporting Issues
- Use the issue templates
- Provide reproducible code examples
- Include system information

## Submitting Changes
- Fork the repository
- Create a feature branch
- Add tests for new functionality
- Ensure all tests pass
- Submit a pull request

## Development Setup
```bash
git clone https://github.com/project-genesis-nebula/Shenqu-Algorithm
cd Shenqu-Algorithm
pip install -e .
pytest tests/
```

Code Standards

路 Follow PEP 8 style guide
路 Include type annotations
路 Write comprehensive docstrings
路 Maintain 100% test coverage
